# Research Notes

This folder contains **working research notes** that support the HTP redo design (`docs/design/`).

Principles:
- Research notes can be opinionated and exploratory.
- Claims that affect the design should be backed by either: (a) code pointers, (b) measured evidence, or (c) clearly
  labeled hypotheses.

Local-only large checkouts (not committed):
- `references/` (gitignored), e.g. Triton source, papers/PDFs, experimental repos.

Suggested checkouts (examples):

```bash
git clone --depth=1 https://github.com/triton-lang/triton.git references/triton
git clone --depth=1 https://github.com/tile-ai/tilelang.git references/tilelang
```
