# IntelliC Story

IntelliC aims to be a compiler stack that is simultaneously human-friendly and
LLM-friendly.

The name means "intelligent compiler"; use `intellic` for package/module names
and `ic` only as a short local abbreviation.

The clean rebuild starts from one principle: every major compiler representation
must be explainable, inspectable, replayable, and verifiable before it becomes
an implementation dependency.
