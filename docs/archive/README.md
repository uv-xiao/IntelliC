# Archive

`docs/archive/` preserves closed-PR historical artifacts that should remain
available but are no longer active project state.

Each PR archive should use a stable, descriptive folder name such as
`pr-69-adopt-high-level-intellic-compiler-architecture/`. When a PR recorded
human wording under `docs/in_progress/human_words/`, move those records into
that PR archive under `human_words/`.
